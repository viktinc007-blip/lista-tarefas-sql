# Mini Projeto SQL - Lista de Tarefas

Projeto de estudo usando **SQLite** para demonstrar operações básicas de banco de dados.

## Funcionalidades
- Criar tabela de tarefas
- Inserir tarefas
- Consultar tarefas
- Atualizar tarefas (marcar como concluída)
- Excluir tarefas

## Como usar
1. Abra o terminal.
2. Entre no SQLite com `sqlite3 tarefas.db`.
3. Copie e cole os comandos do `script.sql`.
4. Use `SELECT * FROM tarefas;` para ver os registros.

## Observações
- Cada comando no script é comentado para facilitar o entendimento.
- Mesmo sem rodar o banco, dá pra entender a lógica só lendo os comentários.
